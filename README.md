# บ้านมวยไทย (Ban Muay Thai)

เว็บไซต์ตัวอย่าง “บ้านมวยไทย” สำหรับนำเสนอโปรแกรมมวยล่าสุด ข่าวมวย บทวิเคราะห์ และช่องทางติดต่อ LINE โดยออกแบบให้รองรับมือถือเป็นหลักด้วย HTML, CSS และ JavaScript แบบไม่ต้องมีขั้นตอน build

## ไฟล์ในโปรเจกต์

- `index.html` — โครงสร้างหน้าเว็บไซต์และเนื้อหาตัวอย่าง
- `styles.css` — สไตล์โทนแดง–ทองแบบพรีเมียม พร้อม responsive layout
- `script.js` — เมนูมือถือและการไฮไลต์เมนูตามส่วนที่กำลังอ่าน

## วิธีเปิดใช้งาน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้ทันที หรือรันเว็บเซิร์ฟเวอร์แบบง่ายด้วยคำสั่ง:

```bash
python3 -m http.server 8000
```

จากนั้นเข้าใช้งานที่ `http://localhost:8000`

## หมายเหตุ

ข้อมูลโปรแกรมมวย ข่าว และบทวิเคราะห์ในเว็บไซต์เป็นข้อมูลตัวอย่าง สามารถแก้ไขเนื้อหาได้โดยตรงใน `index.html`
